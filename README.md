# logic_formula_validator
This simple program checks if the user-enetered logic fromula is valid.

# Allowed input:
1) alfabetic variables a-z
2) numeric variables (true/false) 0-1
3) parentheses ()
4) operators: (and) &, (or) |, (implication) >, (equivalent) =, (negation) !

# How it works:
the program works on the basis of reduction, where correct expressions are simplified until it is no longer possible.
